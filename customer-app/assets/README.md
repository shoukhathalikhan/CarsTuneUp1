# Assets Folder

This folder contains app assets for the CarsTuneUp customer app.

## Required Assets
- **icon.png** - App icon (1024x1024px recommended)
- **splash.png** - Splash screen (1242x2436px recommended)
- **adaptive-icon.png** - Android adaptive icon (1024x1024px)
- **favicon.png** - Web favicon (48x48px)

## Temporary Solution
The app.json has been updated to remove asset requirements temporarily.
You can add custom assets later by replacing the paths in app.json.
